# Wallawood Tour

An interactive Gemini capsule that demonstrates the features of the [Wallawood](https://github.com/wallawood/wallawood) library.

## Running

```bash
./gradlew run
```

The capsule starts on `gemini://localhost:1965`. Open it in any Gemini client to walk through routing, context injection, security annotations, error handling, and more.

## Requirements

- Java 21+
- A Gemini client (e.g. [Lagrange](https://gmi.skyjake.fi/lagrange/), [Kristall](https://kristall.random-projects.net))
